# あやめ

## サイト概要
ひとり親のためのコミュニティサイト

### サイトテーマ
*ひとり親が孤独にならない
*情報の交換による助け合い
*年齢、月齢、地域が近い人との仲間づくり

### テーマを選んだ理由
自分自身がひとり親になり不安だった為ひとり親の情報交換が出来る場があればいいなと思い選びました。

### ターゲットユーザ
*ひとり親の母
*ひとり親の父
*ひとり親を支援してくれる方

### 主な利用シーン
*ひとり親としてのお得な情報が欲しいとき
*子育てに不安になったとき
*ひとり親としての仲間が欲しいとき
